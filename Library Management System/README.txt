Project Title :-   Library Management System

Technologies :-    Python, SQL (Postgres), Psycopg2 connector

Description :- 

The library management system project is about organizing, managing the library and library-oriented tasks.
It also involves maintaining the database of entering new books and the record of books that have been retrieved 
or issued. The main aim of the project is to provide an ease to handle and automate library management. It also
provides features and interface for maintaining librarians records, student history of issued books and penalties.
